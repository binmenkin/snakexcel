# Snakexcel
A snake game in Excel using VBA.
<br><br>
<img src="show.gif" width="300">

## Installation & Usage
Download the file `snake.xlsm`. Open it, enable macros and play.

### Controls
- `Ctrl + Q` - Enable/Disable the Game.
- `Ctrl + W` - Move Up.
- `Ctrl + S` - Move Down.
- `Ctrl + A` - Move Left.
- `Ctrl + D` - Move Right.
